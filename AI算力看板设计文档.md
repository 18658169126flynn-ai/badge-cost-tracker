# AI算力产业看板 — 制作流程与数据架构

> 创建时间：2026-06-25
> 文件：supply-chain.html（badge-cost-tracker 仓库）
> 外网地址：https://18658169126flynn-ai.github.io/badge-cost-tracker/supply-chain.html
> ⚠️ 此文档为权威设计规范，任何更改必须经 Flynn 确认后执行

---

## 核心原则

1. **数据完全独立** — 不引用胸牌成本看板、Flask看板等任何其他数据源
2. **不交叉引用** — 各看板之间无数据关联
3. **不随意扩展** — 严格按本文档执行，新增功能必须经 Flynn 确认
4. **行业级别数据** — 所有数据必须是行业/市场级别，不涉及具体BOM物料单价

---

## 模块清单（10个）

### 模块① 产业链全景（置顶）
- 内容：算力芯片→存储→封装/PCB→光互连→供电→散热 六环节图示
- 数据来源：硬编码HTML，产业链逻辑固定
- 更新频率：仅当产业链结构变化时手动调整
- 展示逻辑：静态展示，纯视觉引导

### 模块② 覆盖板块标签
- 内容：存储芯片/PCB-CCL/MLCC/CPO/光纤/800V/特气/锂电 8个标签
- 数据来源：硬编码HTML
- 更新频率：新增板块时手动添加
- 展示逻辑：静态标签

### 模块③ KPI概览卡片（5个）
- 内容：存储DRAM价格、MLCC涨幅、光纤景气度、CPO状态、800V进展
- 数据来源：手动编辑，来自券商研报和行业资讯（TrendForce/证券时报/wccftech等）
- 更新频率：每周或有重大事件时手动更新
- 展示逻辑：行业级别指标，不涉及具体BOM物料单价

### 模块④ 存储 DRAM 板块详情
- 内容：行业概况（核心逻辑/涨价幅度/供需缺口/国产替代）+ 核心标的表格（6只股票）+ 推荐基金
- 数据来源：手动编辑，行业数据来自TrendForce/美光财报/36氪
- 更新频率：有重大事件（财报/涨价函/产能变化）时更新
- 展示逻辑：左侧文字概述 + 右侧标的表格 + 基金推荐卡片

### 模块⑤ CPO 共封装光学板块详情
- 内容：行业概况 + 核心标的（6只） + 推荐基金
- 数据来源：手动编辑，来自英伟达发布会/证券时报/东方财富
- 更新频率：产品发布/股价异动时更新
- 展示逻辑：同模块④

### 模块⑥ 光纤光缆板块详情
- 内容：行业概况 + 核心标的（5只） + 推荐基金
- 数据来源：手动编辑，来自证券时报/新闻联播/券商研报
- 更新频率：季报/大订单/价格异动时更新
- 展示逻辑：同模块④

### 模块⑦ 800V 高压直流供电板块详情
- 内容：行业概况 + 核心标的（6只） + 推荐基金
- 数据来源：手动编辑，来自钛媒体/COMPUTEX/英伟达发布会
- 更新频率：技术验证/概念股异动时更新（相对低频）
- 展示逻辑：同模块④

### 模块⑧ 投资机会（两个时间维度）
- 内容：未来7-30天重点（6只）+ 未来1-3个月持续关注（8只）
- 数据来源：手动编辑，综合各板块分析 + 机构共识
- 更新频率：市场风格切换/重大催化事件时调整
- 展示逻辑：左侧重点标的含催化剂+风险提示，右侧中长期标的含关注节点

### 模块⑨ 美股AI算力板块走势（自动化）
- 内容：S&P/纳斯达克/道琼斯 + 原油 + 5组美股个股 + 综合研判
- 数据来源：**us_market.json**（Yahoo Finance，前端JS动态渲染）
- 更新频率：每个交易日自动（Cron: 周二-周六 7:30）
- 展示逻辑：浏览器fetch JSON → JS动态生成指数卡片+板块表格
- Cron任务：美股AI算力数据更新（a3971569）

### 模块⑩ 主力资金流向（自动化）
- 内容：AI相关概念板块资金流入 + 核心行业板块资金流入
- 数据来源：**capital_flow.json**（前端JS动态渲染）
  - 主数据源：东方财富 push2 API
  - 降级数据源：新浪财经 API
  - 最终降级：Cron agent 用 web_search 手动构造
- 更新频率：每天早8点自动（Cron: e82edeef）
- 展示逻辑：浏览器fetch JSON → JS动态生成表格 → 资金信号标签

### 模块⑪ 风险提示（4个维度）
- 内容：估值风险/光纤CPO持续性/800V尚早/AI需求验证
- 数据来源：手动编辑
- 更新频率：风险事件变化时更新
- 展示逻辑：红/橙/绿色边框卡片

### 模块⑫ 关键资讯（自动化）
- 内容：各板块涨价/事件新闻摘要
- 数据来源：**industry_news.json**（web_search，前端JS动态渲染）
- 更新频率：每个交易日自动（Cron: 周一-周五 8:00/18:00）
- 展示逻辑：浏览器fetch JSON → JS动态生成新闻表格
