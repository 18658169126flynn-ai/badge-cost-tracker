> 文件：index.html（badge-cost-tracker 仓库）
> 外网地址：https://18658169126flynn-ai.github.io/badge-cost-tracker/
> ⚠️ 此文档为权威设计规范，任何更改必须经 Flynn 确认后执行

---

## 核心原则

1. **数据完全独立** — 不引用AI算力看板、Flask看板等任何其他数据源
2. **单一数据源** — trendData 是唯一的物料价格数据源，bomItems 通过 trendKey 自动关联
3. **不随意扩展** — 严格按本文档执行，新增功能必须经 Flynn 确认
4. **BOM级别数据** — 所有数据基于胸牌记录仪BOM V2.7，追踪特定物料的采购价格

## 模块清单（8个）

### 模块① KPI概览卡片（3个）
- 内容：当前BOM总成本、整机报价¥295、最新整机成本参考
- 数据来源：**JS计算**，基于 trendData 最后一条 + baselineBOM(245.43) + 组装/包材/利润常量
- 更新频率：随 trendData 更新自动变化
- 展示逻辑：calcBOM() 算BOM → calcFull() 算整机 → 与基准对比显示涨幅百分比

### 模块② 成本预测（3个时间维度）
- 内容：1个月/3个月/6个月后整机成本预测，含保守/基准/激进三种场景
- 数据来源：**JS实时计算**，基于 trendData 最后两点斜率线性外推 × 场景系数
- 更新频率：随 trendData 自动变化
- 展示逻辑：predict(months, scenario) → calcFull() → 三场景并列显示

### 模块③ 趋势图表（2个）
- 内容：整机总成本趋势（含预测虚线）+ 关键物料价格走势（双Y轴）
- 数据来源：**Chart.js渲染**，数据来自 trendData 数组
- 更新频率：随 trendData 自动变化
- 展示逻辑：左图=整机成本+三场景预测+基准线，右图=6条物料价格曲线

### 模块④ BOM物料成本拆解（14项）
- 内容：分类/物料/基准报价/当前市价/涨幅/成本影响/风险等级 + 备注说明
- 数据来源：**bomItems 数组**，有 trendKey 的项自动从 trendData 取值，无 trendKey 的项手动维护 current
- 更新频率：trendKey 项随 trendData 自动变化，其他项手动更新
- 展示逻辑：遍历数组 → 计算涨幅 → 生成表格行 → 风险等级着色（红/橙/绿）

### 模块⑤ 供应风险卡片（8个）
- 内容：存储/锂电/PCB/MLCC/主控/TF卡/水墨屏/摄像头 各物料风险评估
- 数据来源：**硬编码JS数组 risks**，每项含 title/cls/text
- 更新频率：手动更新风险描述和等级
- 展示逻辑：遍历数组 → 生成红/橙/绿色左边框卡片

### 模块⑥ 关键涨价资讯（8-10条）
- 内容：各物料涨价/缺货情况的新闻摘要
- 数据来源：**硬编码JS数组 news**，每项含 comp/text/src/time
- 更新频率：手动更新，保持1周内最新新闻
- 展示逻辑：遍历数组 → 生成四列表格（物料/资讯/来源/时间）
- ⚠️ **严格只放物料涨价/缺货信息**，不放资本市场动态（IPO/股价涨幅等）

### 模块⑦ 基础参数（常量）
- baselineBOM = 245.43 — BOM基准成本
- assembleCost = 10 — 组装费
- packagingCost = 15 — 包材费
- profitMargin = 20 — 利润
- baselineFull = 295 — 整机报价基准
- tfcardCost = 17.5 — TF卡客供价
- 更新频率：仅BOM版本变更时调整

### 模块⑧ 趋势数据（trendData 数组）
- 内容：历史时间点的6项物料价格（存储2+16/存储1+8/PCB/阻容/64G TF/32G TF）
- 数据来源：**硬编码JS数组**，来自电子料报价趋势表，由 Flynn 提供价格
- 更新频率：手动新增数据点
- ⚠️ **这是唯一的价格数据源**，bomItems 中有 trendKey 的项自动从此取值

## 数据流关系

```
trendData[] ──→ calcBOM() ──→ KPI卡片（模块①）
            ──→ calcFull() ──→ KPI卡片 + 预测 + 图表（模块①②③）
            ──→ predict()  ──→ 预测卡片 + 图表虚线（模块②③）
            ──→ bomItems[trendKey] ──→ BOM拆解表格（模块④）

bomItems[]  ──→ BOM拆解表格（模块④）
risks[]     ──→ 风险卡片（模块⑤）
news[]      ──→ 新闻表格（模块⑥）
常量        ──→ 基准线/基准报价（模块①③⑦）
```

---

## 更新价格操作规范

当 Flynn 提供最新物料价格时，**只需更新 trendData 一处**：

1. 在 trendData 数组末尾新增一条数据点
2. bomItems 中有 trendKey 的4项（存储/TF卡/PCB/MLCC）自动同步
3. KPI/预测/图表自动重新计算
4. 无 trendKey 的10项（主控/水墨屏/电池/摄像头等）需单独确认是否更新

5. ❌ 不得在未经Flynn确认的情况下修改数据源或更新逻辑
6. ❌ 不得让 trendData 和 bomItems 存在重复维护的数据
